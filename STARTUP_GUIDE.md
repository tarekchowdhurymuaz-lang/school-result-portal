# School Result Portal - স্টার্টআপ গাইড

## 🚀 দ্রুত শুরু করুন

### ১. ইনস্টলেশন

```bash
# প্রজেক্ট ডিরেক্টরিতে যান
cd school-result-portal

# ভার্চুয়াল এনভায়রনমেন্ট তৈরি করুন
python -m venv venv

# Windows এ সক্রিয় করুন
venv\Scripts\activate

# Mac/Linux এ সক্রিয় করুন
source venv/bin/activate

# ডিপেন্ডেন্সি ইনস্টল করুন
pip install -r requirements.txt
```

### २. অ্যাপ্লিকেশন চালান

```bash
python app.py
```

ব্রাউজারে খুলুন: `http://localhost:5000`

## 📊 Excel ফাইল তৈরি করুন

আপনার Excel ফাইলে এই কলামগুলো থাকা দরকার:

| ক্লাস | শিফট | পরীক্ষা | রোল | নাম | বাংলা | ইংরেজি | গণিত |
|------|------|--------|------|------|-------|--------|-------|
| ६    | সকাল | মাসিক १ | १    | রহিম | ८५   | ७५     | ९०   |
| ६    | সকাল | মাসিক १ | २    | সালমা | ९०   | ८८     | ९२   |

**গুরুত্বপূর্ণ:**
- প্রথম পাঁচটি কলাম (ক্লাস, শিফট, পরীক্ষা, রোল, নাম) থাকা বাধ্যতামূলক
- বাকি সব কলাম স্বয়ংক্রিয়ভাবে বিষয় হিসেবে ধরা হবে
- কলামের নাম যেকোনো ভাবে হতে পারে (বাংলা, ইংরেজি, মিশ্রিত)

## 🔧 API এন্ডপয়েন্টস

### POST `/api/upload`
Excel ফাইল আপলোড করুন
```json
{
  "file": "<Excel file>"
}
```

### GET `/api/classes`
সব ক্লাস পান
```json
{
  "success": true,
  "classes": ["६", "७", "८"]
}
```

### GET `/api/shifts/<class>`
নির্দিষ্ট ক্লাসের শিফট পান

### GET `/api/exams/<class>/<shift>`
পরীক্ষা পান

### GET `/api/students/<class>/<shift>/<exam>`
শিক্ষার্থী তালিকা পান

### POST `/api/result`
ফলাফল পান
```json
{
  "class": "६",
  "shift": "সকাল",
  "exam": "মাসিক १",
  "roll": "१"
}
```

## ⚙️ কনফিগারেশন

```python

MAX_CONTENT_LENGTH = 50 * 1024 * 1024  # 50MB

# Debug মোড
DEBUG = True  # Development এর জন্য
DEBUG = False # Production এর জন্য

# Session টাইমআউট
PERMANENT_SESSION_LIFETIME = timedelta(days=7)
```

## 🐛 সমস্যা সমাধান

### Excel ফাইল আপলোড হচ্ছে না
- ✅ ফাইল .xlsx বা .xls ফরম্যাটে আছে কিনা চেক করুন
- ✅ প্রয়োজনীয় কলাম (ক্লাস, শিফট, পরীক্ষা, রোল, নাম) আছে কিনা চেক করুন
- ✅ ফাইল সাইজ 50MB এর কম আছে কিনা চেক করুন

### ড্রপডাউন খালি দেখাচ্ছে
- ✅ Excel ফাইল সঠিকভাবে আপলোড হয়েছে কিনা চেক করুন
- ✅ ব্রাউজার কনসোলে এরর দেখতে F12 চাপুন

### শিক্ষার্থী খুঁজে পাওয়া যাচ্ছে না
- ✅ সঠিক ক্লাস, শিফট, পরীক্ষা নির্বাচন করেছেন কিনা
- ✅ রোল নম্বর বা নাম সঠিকভাবে টাইপ করেছেন কিনা
- ✅ Excel ফাইলে শিক্ষার্থী আছে কিনা চেক করুন
